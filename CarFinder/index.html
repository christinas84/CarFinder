Welcome to Carfinder!

This Car-Finder Web API application was built using ASP.NET C# and SQL server T-SQL. Providing the user with the most accurate information utilizing the National Highway Traffic Safety Administration Recall API and the Bing Image Search API.

The TNGGrid pluggin, Bootstrap CSS and AngularJS was used resulting in a fully responsive web application.
